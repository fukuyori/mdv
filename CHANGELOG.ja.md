# 変更履歴

このプロジェクトの主な変更点を記録しています。

書式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に基づき、
バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に従います。

[English](CHANGELOG.md) | 日本語

## [0.3.1] - 2026-07-08

### 追加

- 複数ウインドウ対応: タブを右クリックして「新しいウインドウで開く」を
  選ぶと、そのタブだけを新しいウインドウへ移動できる(タブが2つ以上ある
  ときのみ表示)。
- タブをタブバーの外へドラッグすると、そのタブを新しいウインドウとして
  切り離せる。ドラッグ中はタブの半透明のプレビューがカーソルに追従する。
- タブをドラッグして別の mdv ウインドウへドロップすると、新しいウインドウ
  を作らずそちらへタブを移動する。移動元のウインドウの最後の1タブを
  移動した場合は、そのウインドウは閉じてウインドウが1つだけになる。

## [0.3.0] - 2026-07-08

### 追加

- タブ機能: 複数の Markdown ドキュメントを同時に開いて編集可能に。
  各タブは独立したエディタ・アウトライン・プレビューを持つ。タブの
  新規作成・閉じる(未保存の変更がある場合は確認ダイアログを表示)・
  Ctrl+Tab / Ctrl+Shift+Tab での切り替えに対応。
- すでに開いているファイルを再度開こうとした場合、重複したタブを
  作らずに既存のタブへ切り替えるように。
- コマンドライン引数で複数ファイルを指定した場合、およびウィンドウへ
  複数ファイルをドラッグ&ドロップした場合(ドラッグ&ドロップでの
  ファイルオープンは新機能)、それぞれ別タブとして開くように対応。
- 翻訳はバックグラウンドのタブでも継続。あるタブを原文表示に戻しても、
  同じ内容を翻訳中の他のタブには影響しない。
- タブが開いているファイルがディスク上で変更された(他のプログラムに
  よる編集など)ことを検知し、再読み込みするか確認するように。表示中の
  タブは即座に、バックグラウンドのタブは選択した時点で確認する。未保存の
  編集がある状態で再読み込みを見送った場合、終了時の保存確認ダイアログで
  「保存」が外部の変更を上書きし「破棄」が外部の変更を残すことを明示する
  ように変更。

## [0.2.2] - 2026-07-08

### 修正

- macOS: プレビューペインが常に真っ白になり、アクセシビリティクライアント
  (VoiceOver や Karabiner など)がウィンドウを照会すると起動直後に
  クラッシュすることがある問題を修正。署名済みアプリに同梱の
  QtWebEngineProcess ヘルパーが Homebrew の Qt ライブラリを絶対パスで
  参照したままだったため、Chromium レンダラプロセスが起動できなかった。
  macOS パッケージングスクリプトがバンドル内の全ライブラリ参照を
  @rpath に書き換え、不足していた画像フォーマット依存ライブラリ
  (libjpeg・libwebp)を同梱するように変更。

## [0.2.0] - 2026-07-06

### 追加

- ローカルの [Ollama](https://ollama.com) サーバーによるプレビューの翻訳機能。
  プレビュー上部のボタンまたは新設の「翻訳」メニューから、
  「原文」「対訳」(原文と訳文を交互に表示)「翻訳」の3表示を切り替え可能。
- 翻訳の設定ダイアログ: Ollama エンドポイント、モデル(サーバーから
  インストール済み一覧を取得)、翻訳先言語(日本語/English)、
  同時リクエスト数(1〜8)。
- ブロック単位の翻訳と逐次表示: 訳文は到着したブロックから順次表示され、
  ブロック単位でキャッシュされるため、編集時は変更したブロックだけが
  再翻訳されます。
- 表示位置優先の翻訳順: 現在表示中の位置から下のブロックを読み順で優先し、
  スクロールに合わせて待機キューを並べ替えます。
- ブロック単位のエラー処理: 翻訳に失敗したブロック(サーバーエラー、
  タイムアウト、空応答)は対訳表示で「(翻訳失敗)」マーカー付きの原文表示と
  なり、残りの翻訳は継続します。全体を中止するのはサーバーに接続できない
  場合のみです。
- Linux 向けリリースビルド・パッケージングスクリプト(deb / tarball)。
- Linux でのアプリケーションアイコンの埋め込み。

### 修正

- Linux でフォントダイアログが文字化けする問題。
- macOS リリース署名スクリプトの修正。
